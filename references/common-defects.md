# DV 测试报告常见缺陷库

按严重度归类的历史高频问题。用途：审核时对照排查；发现命中缺陷库的问题时，可在审核报告中引用缺陷编号（如 CD-S03）。此库应随实际审核经验持续补充。

## 严重（直接颠覆报告有效性或结论可信度）

| 编号 | 缺陷 | 典型表现 |
|---|---|---|
| CD-S01 | 结论与数据矛盾 | 单项不合格但总结论"通过"；文字"无异常"但数据超限 |
| CD-S02 | 试验漏项 | DV大纲/标准要求的项目在报告中缺失，且无偏离说明 |
| CD-S03 | 设备校准失效 | 校准有效期早于试验日期；使用超期设备 |
| CD-S04 | 样品不可追溯 | 编号前后不一、无批次信息，数据无法对应到具体样品 |
| CD-S05 | 判定准则错误 | 用错标准版本/条款；判定方向用反（≤/≥） |
| CD-S06 | 数据誊抄错误迹象 | 汇总表数值与单项记录不符；多页同一数据互不相同 |
| CD-S07 | 签署无效 | 缺批准签署；编制与批准同人 |
| CD-S08 | 试验日期倒挂 | 试验日期早于委托/送样日期，或晚于报告签发日期 |

## 一般（规范性/可信度问题，不直接颠覆结论）

| 编号 | 缺陷 | 典型表现 |
|---|---|---|
| CD-G01 | 引用作废标准年号 | 如 GB 38031 引用 2015 而非现行版本 |
| CD-G02 | 同一标准年号前后不一 | 一处 2020、一处 2015 |
| CD-G03 | 临界值未提示 | 实测值贴近限值（<5% 裕度）但报告未作任何说明 |
| CD-G04 | 缺测量不确定度说明 | 定量项目声明了精度却未给不确定度（如客户要求） |
| CD-G05 | 曲线图要素不全 | 坐标轴无单位、无图例、无图号 |
| CD-G06 | 照片无标识 | 无样品编号/日期/试验项目名称 |
| CD-G07 | 占位符未清 | "××"、"待补充"、空单元格、"—"无说明 |
| CD-G08 | 环境条件缺失 | 未记录温湿度，或记录值超出标准规定范围 |
| CD-G09 | 失效项无闭环 | 不合格项无复测/整改/让步说明 |
| CD-G10 | 报告编号不一致 | 封面、页眉、汇总页编号互不相同 |

## 建议（可读性/习惯问题）

| 编号 | 缺陷 | 典型表现 |
|---|---|---|
| CD-J01 | 单位写法不统一 | kwh/kWh 混用；℃ 与 °C 混用；全角％与半角%混用 |
| CD-J02 | 有效数字随意 | 同一参数小数位数忽多忽少 |
| CD-J03 | PPT 图表无来源 | 汇报页数据无报告编号/表号出处 |
| CD-J04 | 页眉页脚信息缺失 | 缺报告编号或页码 |
| CD-J05 | 修订记录缺失 | 换版报告无修订历史表 |
| CD-J06 | 百分比无基准 | 容量保持率未说明相对初始值还是额定值 |
