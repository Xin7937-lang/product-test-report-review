# 公司报告模板档案

用途：把公司 DV 报告模板的结构要求登记在这里，审核时核对受审报告的**结构完整性**。
本文件是空白档案：把下面"必备章节"中的示例条目替换为公司真实章节名即生效。

- 脚本（CHECK-8）只检查"必备章节"关键词是否出现在报告中，输出缺失线索
- 审核 agent 还会用本文件做语义判断（章节顺序、必备表格、签署结构）
- 不填写（只有示例条目）时，相关检查自动跳过，不影响其它审核
- 含"（示例）"的行不参与检查
